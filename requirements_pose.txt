# ========================================================================================
# Jetson-Specific Notes
# ========================================================================================
#
# 1. PyTorch Installation:
#    PyTorch should be installed via NVIDIA's pre-built wheels:
#    https://forums.developer.nvidia.com/t/pytorch-for-jetson/72048
#
#    Example for JetPack 5.x (Ubuntu 20.04):
#    wget https://nvidia.box.com/shared/static/[wheel_file].whl
#    pip3 install torch-*.whl
#
# 2. TorchVision Installation:
#    Build from source or use NVIDIA's wheels:
#    sudo apt-get install libjpeg-dev zlib1g-dev libpython3-dev libopenblas-dev libavcodec-dev libavformat-dev libswscale-dev
#    git clone --branch <version> https://github.com/pytorch/vision torchvision
#    cd torchvision
#    export BUILD_VERSION=<version>
#    python3 setup.py install --user
#
# 3. OpenCV:
#    Often pre-installed with JetPack. Check with:
#    python3 -c "import cv2; print(cv2.__version__)"
#
# 4. Ultralytics YOLOv8:
#    Should work out of the box with:
#    pip3 install ultralytics
#
# 5. Memory Optimization:
#    - Use batch_size=8-16 for Xavier AGX (32GB)
#    - Enable CUDA if available
#    - Use torch.cuda.amp for mixed precision training
#
# 6. Performance Tips:
#    - Set CUDA device: export CUDA_VISIBLE_DEVICES=0
#    - Enable max power mode: sudo nvpmodel -m 0
#    - Enable max clock speeds: sudo jetson_clocks
#
